# Archive de changes completadas

Este directorio contiene las changes que pasaron por todo el workflow (proposal → spec → design → tasks → apply → verify) y fueron archivadas.

Convención de nombre: `YYYY-MM-DD-{change-name}/`

Las changes archivadas son **read-only**: documentan qué se hizo, cuándo, por qué, y con qué evidencia. Si necesitás revisar un change archivado para hacer algo similar, **NO lo edites** — usalo como referencia para hacer una change nueva.

Las specs delta de la change archivada YA fueron mergeadas a `../../specs/{domain}/spec.md` (source of truth) durante `sdd-archive`.
