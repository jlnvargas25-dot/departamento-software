## Cómo usar este archivo

- **Agregar** deuda nueva cuando aparezca (durante el paso 5 del cierre de sesión)
- **Priorizar** por impacto x urgencia
- **Resolver** marcando con ✅ y fecha + razón
- **Archivar** items resueltos en sección "Resueltos" al final del archivo
- **Formato** cada item:

```
### DEUDA-<ID> — <Título corto>
**Status**: 🔴 ABIERTA / 🟡 EN PROGRESO / ✅ RESUELTA
**Severidad**: bloqueante / alta / media / baja
**Descubierto**: YYYY-MM-DD
**Owner**: quién la va a resolver

Descripción del problema.

**Resolución propuesta**:
- (Pasos para resolver)

**Costo estimado**: (tiempo)
**Beneficio esperado**: (qué cambia cuando se resuelva)
```
